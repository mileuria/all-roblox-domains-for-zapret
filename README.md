### Roblox домены для Zapret
***
- Roblox лично у меня перестал загружать картинки и игра стала работать ужасно. В этом репозитории собраны (почти) все домены, связанные с Roblox, которые мне пришлось добавить в Zapret для нормальной игры.

### Как добавить домены в Zapret
***
- Откройте папку с Zapret и перейдите в папку `lists`
- Найдите list-google.txt ИЛИ создайте новый .txt файл в этой папке — имя файла `list-roblox.txt`, необязательно
- Вставьте домены из этого репозитория в файл и сохраните (Ctrl+S)
- Запустите `service.bat`
- Включите `Game Filter`
- Установите `ipset` в значение `any`
  - Если вы это сделали, Zapret может попросить перезапустить сервис
- В диалоговом меню введите 1, нажмите Enter и выберите 1 ALT либо второй ALT; при необходимости проверьте все ALT
- Выйдите из меню, введя 0 и нажав Enter
- Проверьте игру

### Важные заметки
***
- Рекомендуется создать резервную копию исходного list-google.txt перед изменениями
- Я использую Zapret версии 1.9.0b; интерфейс в 1.9.4 может отличаться
- Если игра всё ещё некорректно работает, временно отключайте добавленные записи по одной, чтобы найти конфликтный домен. Zapret не сразу применяет все настройки, через несколько секунд.
